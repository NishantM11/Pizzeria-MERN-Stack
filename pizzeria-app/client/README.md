# Pizzeria Web Application

This is a MERN stack Pizzeria web application that allows users to browse a menu of pizzas, add items to their cart, and place orders. 

## Client-Side

The client-side of the application is built using React and is located in the `client` directory. Below are the key components and pages:

- **Components**:
  - `Header.js`: Renders the navigation bar.
  - `Menu.js`: Displays the list of available pizzas.
  - `Cart.js`: Shows items added to the cart and allows checkout.
  - `OrderForm.js`: Handles user input for placing an order.

- **Pages**:
  - `Home.js`: The landing page of the application.
  - `Menu.js`: Displays the menu page with available pizzas.
  - `Orders.js`: Shows the user's past orders.

- **Services**:
  - `api.js`: Contains functions for making API calls to the server.

- **Main Application**:
  - `App.js`: Sets up routing and renders the main layout.
  - `index.js`: Entry point of the client application.

## Getting Started

To get started with the client-side application, follow these steps:

1. Navigate to the `client` directory.
2. Install the dependencies using npm:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm start
   ```
